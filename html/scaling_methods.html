Scaling methods¶

The module scaling_methods provides various functions for calculating scaling properties of time series, in particular calculating: - lag-1 autocorrelation function - DFA scaling exponent - Power Spectrum scaling exponent

And, in particular, each of these calculated in a sliding window.

scaling_methods.acf(z: numpy.ndarray, lag: int = 1)¶

Calculates the autocorrelation function of a time series \(z\).

Parameters

  • z – Time series

  • lag – The autocorrelation lag

Returns

float: autocorrelation function of a time series z

scaling_methods.acf_scaling(z: numpy.ndarray)¶

Calculates the autocorrelation scaling exponent of a time series \(z\).

This calculation involves calculating the ACF with lag \(l\) for \(l = 10, 11, 12, ..., 100\) and then calculating the negative gradient of the log-log plot \(l\) against ACF.

Parameters

z – The time series

Returns

float: autocorrelation scaling exponent of a time series z

scaling_methods.acf_sliding(t: numpy.ndarray, z: numpy.ndarray, window_size: int = 200, lag: int = 1, increment: int = 1)¶

For a time series \(z(t)\) we select a slice of the data defined by a sliding window e.g. for window_size=200 the first data slice would be z[0:200].

We then measure the ACF (usually lag-1 ACF) of this slice and progress the window by an increment, e.g. for increment=1 the second data slice would be z[1:201].

This function returns y_out: the series of these ACF values, and t_out, the corresponding time variable.

Parameters

  • t – time variable

  • z – dependent variable

  • window_size – length of the sliding window

  • lag – the lag used in the ACF calculation

  • increment – the increment by which the window is progressed at each step

Returns

numpy.ndarray: t_out, y_out

scaling_methods.dfa_sliding(t: numpy.ndarray, z: numpy.ndarray, window_size: int = 200, order: int = 2, no_segment_lengths: int = 8, increment: Optional[int] = None)¶

For a time series \(z(t)\) we select a slice of the data defined by a sliding window e.g. for window_size=200 the first data slice would be z[0:200].

We then measure the DFA exponent of this slice and progress the window by an increment, e.g. for increment=1 the second data slice would be z[1:201]. The increment, if not user-specified, is set at \(0.2\) times the square root of the series length.

This function returns y_out: the series of these ACF values, and t_out, the corresponding time variable.

Parameters

  • t – time variable

  • z – dependent variable

  • window_size – length of the sliding window

  • order – the polynomial order used in the DFA exponent calculation

  • increment – the increment by which the window is progressed at each step

Returns

numpy.ndarray: t_out, y_out

scaling_methods.pse_sliding(t: numpy.ndarray, z: numpy.ndarray, window_size: int = 200, binning: bool = True, window_limits: (<class 'float'>, <class 'float'>) = (-2, -1), increment: int = 1)¶

For a time series \(z(t)\) we select a slice of the data defined by a sliding window e.g. for window_size=200 the first data slice would be z[0:200].

We then measure the power spectrum scaling exponent of this slice and progress the window by an increment, e.g. for increment=1 the second data slice would be z[1:201].

This function returns y_out: the series of these ACF values, and t_out, the corresponding time variable.

Parameters

  • t – time variable

  • z – dependent variable

  • window_size – length of the sliding window

  • binning – whether or not to use logarithmic binning on the periodogram as part of the PS exponent calculation

  • increment – the increment by which the window is progressed at each step

Returns

numpy.ndarray: t_out, y_out

scaling_methods.pse_test()¶
If we have AR(1) models with different parameters mu

and evaluate the pse value beta for each

Then the mu and beta values should follow the curve:

mu = b - sqrt(b^2 - 1) where b = [cos(0.2*pi) - 10^beta * cos(0.2*pi)] / [1 - 10^beta]
