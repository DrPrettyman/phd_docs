The individual modules in tippingpoints may be used to carry out certain techniques or methods developed during this project. For example, running scaling_methods.pse_sliding(t, z) will calculate the sliding-window power spectrum scaling exponent for time series z.

In the modules figures_c1, figures_c2 there is the code ready to reproduce particular figures and tables from the thesis.
