Numerical methods¶

The module numerical_methods provides the Euler_Maruyama method and the Milstein method to integrate a stochastic equation of the form \(dZ = a(Z)dt + \sigma dW\)

numerical_methods.dW(delta_t)¶

” Random sample normal distribution

numerical_methods.eulermaruyama(a, sigma: float, z_0: float = 0, t_0: float = 0, t_end: float = 1, n: int = 1000, ensemble_size: int = 1, radius: tuple = (- 10, 10))¶

Integrates a stochastic equation of the form \(dZ = a(Z)dt + \sigma dW\) according to the Euler-Maruyama method

Returns the time variable ts and the integrated series ys.

Parameters

  • a – function of t and z returning a single value

  • sigma – standard deviation of stochastic noise

  • z_0 – initial condition in z

  • t_0 – start time

  • t_end – end time

  • n – length of output. The integration time delta is therefore defined as (t_end - t_0) / n

  • ensemble_size – for ensemble size \(m\) the output ys will be a \(n\) by \(m\) array

  • radius – defines a range constraining the integration. The integration stops if ` y < radius[0]` or y > radius[1] in order to prevent blow-up

Returns

numpy.ndarray: ts, ys``
